# sandbox_
오픈소스 샌드박스 게임

## sandbox_ 는 뭔가요?
sandbox_ 는 앞에서 말했다 싶이 오픈 소스 샌드박스 게임이에요!

다양한 커스텀 기능(개발중)이 가장 큰 핵심이라고 할수 있죠.

## 라이선스
최소한의 크래딧이라도 남겨주시면 감사하겠습니다.
